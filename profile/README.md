## Local-first by design

Your keys, configs, and run history stay on **your** machine. Only anonymized, aggregate metrics ever leave — no prompts, no credentials, no PII. We built Invoked to pass enterprise security review, not just to look secure. → [Security](https://invoked.ai/security)

## What you can build

| | |
|---|---|
| **🧩 Surfaces** | Connect MCP servers, tool sets, and APIs. Switch between them with a keystroke — each fully isolated. |
| **🛠️ Tools** | Browse, configure, and compose exactly the capability surface an agent needs. |
| **🧠 Harnesses** | Wire agents into flows on a canvas, run them in dependency order with phase-scoped guardrails, and evaluate every change against a baseline. |
| **♻️ Skills** | Turn any winning run into a reusable, versioned skill — pin a model, lock a tool set, share with your team. |
| **🔍 Run history & evals** | Full trace for every run: reasoning, tool calls, tokens, timing. Compare runs, swap models, score with an LLM judge. |
